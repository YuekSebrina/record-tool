# 收藏夹 MVP 产品说明

## 产品定位

“拾羽”是一个私人的书影音收藏夹，聚焦作品搜索、收藏、评分和阅读/观看进度管理，不承担随记或待办管理。

固定收藏分类：

- 书籍
- 动漫
- 电影
- 剧集

## 核心流程

```text
选择分类
→ 输入作品名称
→ 联网搜索豆瓣
→ 查看封面和基础资料
→ 一键收藏
→ 在收藏页分类查看或删除
```

## 当前边界

- 收藏数据只保存在小程序本地。
- 搜索和封面由 Flask 云托管服务代理。
- 不需要豆瓣 API Key。
- 搜索结果未提供的评分、简介等字段会在打开条目详情时按需补充。
- 暂不支持登录、跨设备同步、自定义收藏夹、拖动排序和手动补充资料。

## MVP 验收

- 四个固定分类都可以发起搜索。
- 书籍走豆瓣图书搜索，动漫、电影和剧集走豆瓣影视搜索。
- 搜索结果可以显示代理后的封面。
- 同一分类的同一豆瓣条目不会重复收藏。
- 收藏页可以分类查看数量，并在列表与三列海报宫格间切换。
- 收藏条目支持 1–5 星评分和自动计算百分比的阅读/观看进度。
- 收藏、评分和页面切换提供轻震反馈。
- 删除收藏前必须二次确认。
- 搜索失败、无结果和空收藏都有明确状态提示。
